# An alternative OpenShift Console plugin template

## Frontend

Features:

* [x] Strict(er) TypeScript configuration
* [x] PatternFly components
* [x] Extraction of localization strings
* [ ] Linter (ESLint)
* [ ] Auto formatter (Prettier)
* [ ] Unit tests
* [ ] E2E tests
